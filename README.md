# Sales Data Analysis
This project analyzes sales data from a CSV file (**Sales.csv**) to understand product performance and revenue.
<br>
This program helps you see
- How much revenue each product made
- Which product earned the most
- Revenue by product type (Fruit, Dairy, Snacks, etc.)
- Revenue by unit type (kg, piece, packet)

## Dataset
The CSV file contains the following columns:
    - **Product**: Product name <br>
    - **Type**: Product category <br>
    - **Quantity**: Number of items sold <br>
    - **Unit**: Unit type (kg, piece, packet) <br>
    - **Rate**: Price per unit
   
## Features
- Calculate **revenue** for each product (**Quantity * Rate**)
- Find the **top product by revenue**
- Summarize **revenue per unit type**
- Summarize **revenue per product type** 

## Visualizations
- **Bar plot** of top 10 products by revenue
- **Pie chart** showing revenue share by product type
